# cyprus-sailboat
本仓库用于整理一篇关于自主帆船与异构浮标信息交互的论文初稿。
这篇论文希望实现并验证一种基于云端 MQTT 的信息交互架构：克罗地亚方浮标将信息发布到云服务器，我方自主帆船上位机订阅对应 Topic，获取并解析浮标信息，从而为自主帆船后续的任务决策和航行控制提供外部信标数据。
论文的核心目标是说明自主帆船不仅能够依赖自身传感器完成航行，还可以通过云端发布/订阅机制接入第三方浮标信息，实现跨机构、异构设备之间的信息互操作。
目前塞浦路斯实验主要用于验证真实第三方浮标信息链路能够打通；后续国内实验计划通过模拟信标发布信息，并让自主帆船完成绕标航行，用于进一步验证该信息交互架构对任务闭环的支撑能力。
